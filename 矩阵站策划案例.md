附件1：百潮荟｜欧美市场品牌内容基础建设方案
—— 基于全宇智能内容协同系统

一、项目背景与总体定位
1.品牌背景
百潮荟当前拥有并运营的核心潮玩 IP / 产品线包括：
A｜唐萌街僵尸派对（Zombie Party）
B｜Lokiki
C｜目及岛
产品整体特征：
原创 IP 方向与角色体系持续扩展
潮玩收藏属性强，盲盒与限量款驱动
视觉与世界观风格偏向 萌系、可爱型与轻怪诞美学，在保持亲和力的同时融入一定的趣味反差，整体更贴近 Z 世代审美与潮流文化语境。
在欧美市场具有明显的差异化审美优势
在品牌调性与商业逻辑上可对标泡泡玛特（POP MART），但在视觉、世界观与角色风格上更适配欧美年轻文化审美与社媒传播语境。

2. 项目总体定位（务必阅读）
本方案定位为：海外品牌内容与信息基础建设项目。
核心目标是为百潮荟在欧美市场建立稳定、可控的品牌信息入口与内容资产沉淀体系，从而在长期降低对单一平台（如 Meta/Google Ads/TikTok/Amazon 等）的依赖风险，并为未来线上业务、自有 IP 与品牌化发展提前完成底层准备。
重要说明：
本项目不作为短期销售或业绩考核项目；其价值以“长期积累、风险配置、信息覆盖”为主，而非即时成交转化。

3. 建设方向（长期目标）
逐步建立百潮荟在欧美市场的品牌搜索认知
搭建可持续沉淀的自然内容与信息覆盖体系
为未来线上零售、自有 IP、海外品牌化提供内容基础与承接载体
长期降低对单一广告或平台流量的依赖风险

二、目标用户画像（欧美核心消费群）
1. 核心用户画像（Primary Persona）
年龄：18–35 岁
地区：美国 / 加拿大 / 英国 / 德国 / 法国
身份特征：
潮玩收藏者
设计、艺术、街头文化爱好者
ACG / 独立 IP / 盲盒文化用户
兴趣标签：
Designer Toys
Art Toys

消费心理特征：
重视 IP 世界观与角色设定
愿意为限量款、设计感与文化标签付费
对“故事感 + 情绪价值”高度敏感

2. 次级用户画像（Secondary Persona）
潮流文化内容消费者（非重度收藏）
礼品购买人群
社交媒体审美型用户（Instagram / TikTok）

三、整体建设策略框架
本项目以“可控承接 + 长期覆盖 + 内容扩散”为核心结构：
用品牌独立站作为官方信息与内容承接入口
用海外内容资源网络作为长期信息覆盖与搜索占位基础
用社交媒体账号作为内容展示、文化传播与互动沉淀渠道
三者通过全宇智能内容协同系统进行统一管理与联动，形成“内容资产持续积累”的长期体系。

四、品牌独立站建设方案
1. 独立站定位
欧美用户的第一品牌信息入口（官方承接）
IP 世界观展示与角色体系呈现
产品信息展示、系列介绍与后续销售准备
品牌故事与内容沉淀
当前阶段以“信息承接与内容建设”为核心，不以成交为目标；销售模块作为后续扩展准备。

2. 核心栏目结构
Home（品牌视觉 + 核心角色展示）
About 百潮荟（品牌与 IP 世界观）
Characters（角色档案）：用于集中展示各 IP 系列中的核心角色设定、形象特征与背景信息，为用户提供清晰、直观的角色认知入口，同时作为后续内容沉淀与搜索信息承载的基础页面
Collections（系列产品介绍）
Blog / News（内容与信息沉淀）
Community（粉丝共创与互动）
Shop（商城模块，阶段性启用）

3. 技术与内容策略
多语言结构：EN / DE / FR（后期逐步扩展）
基础 SEO 结构化设置（便于搜索收录与展示）
与内容资源网络形成互引与引用（信息互联）
与社交平台内容同步展示（内容复用）
